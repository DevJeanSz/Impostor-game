# 🎮 Jogos em Grupo

Bem-vindo ao **Jogos em Grupo**! Uma coleção divertida de jogos para animar suas festas, reuniões com amigos ou quebrar o gelo em qualquer ocasião. Tudo offline, gratuito e direto no seu navegador.

![Jogos em Grupo](https://images.unsplash.com/photo-1511632765486-a01980e01a18?q=80&w=2070&auto=format&fit=crop)

## 🚀 Sobre o Projeto

Este aplicativo foi desenvolvido para ser a solução definitiva para momentos de tédio em grupo. Sem necessidade de internet (após o carregamento inicial), sem anúncios intrusivos e com uma interface moderna e fluida.

O objetivo é simples: **Juntar a galera e se divertir!**

## 🎲 Jogos Disponíveis

Atualmente, contamos com 3 jogos incríveis:

### 👻 1. Quem é o impostor?
*Inspirado no clássico "Spyfall" ou "O Impostor".*

**Como funciona:**
- Todos os jogadores recebem uma palavra secreta, exceto um (ou mais) impostores.
- O objetivo do grupo é descobrir quem é o impostor fazendo perguntas uns aos outros.
- O objetivo do impostor é descobrir qual é a palavra secreta sem ser pego.
- **Destaque:** Milhares de palavras em categorias como Animais, Comida, Lugares, Profissões e muito mais!

### 👉 2. Quem é mais provável
*O jogo perfeito para expor seus amigos (com carinho).*

**Como funciona:**
- Uma pergunta aparece na tela (ex: "Quem é mais provável de sobreviver a um apocalipse zumbi?").
- No "3, 2, 1...", todos apontam para quem acham que mais se encaixa na descrição.
- Prepare-se para risadas e debates acalorados!

### ⚡ 3. Mímica
*O clássico que nunca sai de moda.*

**Como funciona:**
- Divida-se em times ou jogue todos contra todos.
- Um jogador vê uma palavra e deve atuar para que os outros adivinhem.
- Sem falar, sem emitir sons, apenas gestos!

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído com tecnologias modernas para garantir performance e uma ótima experiência de usuário:

- **React** - Para uma interface reativa e rápida.
- **Tailwind CSS** - Para um design bonito e responsivo.
- **Framer Motion** - Para animações suaves e agradáveis.
- **Lucide React** - Para ícones nítidos e leves.
- **Vite** - Para um desenvolvimento ágil e build otimizado.

## 📱 Como Usar

1. Acesse o link da aplicação.
2. Escolha um dos jogos na tela inicial.
3. Siga as instruções de configuração (adicionar jogadores, escolher categorias, etc.).
4. Divirta-se!
